# Documentação dos Scripts

Este repositório contém três scripts Python criados utilizando o GitHub Copilot no VS Code. O Copilot foi utilizado como uma ferramenta de auxílio à programação, gerando o código automaticamente a partir dos comentários iniciais de cada arquivo, que serviram como prompts para a IA.

## O que é o Copilot?
O GitHub Copilot é uma inteligência artificial integrada ao Visual Studio Code que sugere e gera trechos de código automaticamente, com base em comentários, contexto do arquivo e boas práticas de programação. Basta escrever um comentário explicando o que deseja fazer, e o Copilot propõe uma solução em código, acelerando o desenvolvimento e facilitando o aprendizado.

## Scripts presentes

- **concat_dados.py**: Solicita duas informações do usuário e as concatena em uma única string, exibindo o resultado.
- **ope_dados.py**: Recebe dois números e uma operação matemática (+, -, *, /) do usuário, realiza o cálculo e mostra o resultado. Inclui tratamento para divisão por zero.
- **repet_txt.py**: Solicita uma string e um número inteiro, e retorna a string repetida o número de vezes informado, separada por espaços.

Cada script foi gerado a partir de um comentário inicial que descreve o objetivo do código. O Copilot interpretou esse comentário e sugeriu a implementação correspondente, tornando o processo de criação mais rápido e intuitivo.
